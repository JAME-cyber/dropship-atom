## 📐 Règles de publication (anti-échec)

1. **Publier le tweet 1 + le thread en reply** (pas un seul bloc). Le hook seul capte les scroll-eurs pressés ; le thread convertit les intéressés.
2. **Sur LinkedIn, poster le jour de semaine à 8h30 ou 12h15** (pics FR). Pas le week-end.
3. **Ne PAS répondre aux haters** dans les 2 premières heures (l'algorithme pénalise les débats précoces sur les posts neufs). Répondre uniquement aux questions sérieuses après 2h.
4. **Si 0 engagement à H+6** → ne pas relancer le même post. Attendre 48h, reformuler le hook avec un angle différent (ex : « 95 % des dropshippers perdent de l'argent. Voici pourquoi. »).
5. **Le compteur "5 exemplaires" doit être tenu à jour publiquement** : répondre "3/5 restants" sous le post dès qu'une vente entre. La rareté visible convertit plus que la rareté annoncée.
