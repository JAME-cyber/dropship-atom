L'analogie USB est brillante. Mais la leçon de l'USB c'est pas "construis des câbles USB".

- USB → commodité (1$ le câble)
- MCP → commodité (serveur MCP = ~0$ avec templates)
- La valeur = **ce que tu connectes**, pas le connecteur

**Application DropAtom**: Notre pipeline HUNTER→SCOUT→CREATOR ne dépend PAS du protocole. Si MCP meurt demain et que "ACP" (Agent Communication Protocol) le remplace, nos agents restent valables. On est le device, pas le câble.
